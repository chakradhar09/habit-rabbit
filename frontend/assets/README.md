## Design Concept

The logo features:
- **Abstract "H" Mark**: Represents "Habit" with clean, modern lines
- **Growth Indicators**: Three orange dots of decreasing size symbolize progress and habit formation
- **Blue Gradient**: iOS-inspired gradient (#007AFF ‚Üí #5AC8FA) represents trust, consistency, and forward movement
- **Orange Accents**: Energy, achievement, and positive reinforcement (#FF9500)

## Usage Guidelines

### Clear Space
Maintain a minimum padding of **16px** around the logo on all sides.

### Minimum Size
- Full logo: Minimum width 120px
- Icon mark: Minimum 24x24px

### Color Variations
The logo is designed for **dark backgrounds**. For light backgrounds, ensure sufficient contrast by:
- Using the icon mark with a dark background circle
- Adjusting opacity or adding subtle shadows

### HTML Implementation

```html
<!-- Full Logo -->
<img src="assets/logo.svg" alt="Habit Rabbit" width="200" height="60">

<!-- Icon Mark -->
<img src="assets/logo-icon.svg" alt="Habit Rabbit" width="60" height="60">

<!-- Favicon (in <head>) -->
<link rel="icon" type="image/svg+xml" href="assets/favicon.svg">
```
